# 古代哲学（前6世纪–5世纪）
1. 古希腊前苏格拉底哲学（前6世纪–前5世纪）
    1. 泰勒斯、阿那克西曼德、赫拉克利特、毕达哥拉斯、巴门尼德、芝诺等
    2. 核心问题：世界的本原是什么？
    3. 从神话解释转向理性、自然哲学解释，开启了本体论。
2. 古希腊古典时期（前5–前4世纪）
    1. 苏格拉底 → 柏拉图 → 亚里士多德
    2. 柏拉图说: 真实的存在是永恒不变的“理念”，感官世界只是影子
    3. 西方哲学几乎所有领域（形而上学、认识论、伦理学、政治哲学、美学、逻辑学）都在这三位手里奠基。
3. 希腊化-罗马时期（前4世纪–5世纪）
    1. 伊壁鸠鲁学派（快乐主义）、斯多葛学派（顺应自然、理性生活）、怀疑主义、犬儒学派、新柏拉图主义（普罗提诺）
    2. 哲学开始关注个人幸福与生活方式。
4. 中国轴心时代（前6–前3世纪）
    1. 孔子、老子、墨子、庄子、孟子、荀子、韩非子
    2. 儒家、道家、法家、墨家、名家、阴阳家等“诸子百家”
    3. 与古希腊几乎同时出现了高度发达的伦理-政治哲学和形而上学。
5. 印度轴心时代
    1. 奥义书 → 佛教（释迦牟尼）、耆那教、六派哲学（数论、瑜伽、吠檀多等）

# 中世纪哲学（5–15世纪）
1. 西方：基督教哲学（经院哲学）
    1. 奥古斯丁（基督教+新柏拉图主义）→ 安瑟伦 → 阿伯拉尔 → 托马斯·阿奎那（亚里士多德+基督教的集大成者）→ 邓·司各脱、奥卡姆
    2. 主要任务：用希腊理性为基督教信仰辩护（信仰与理性的关系、本体论证明上帝存在）
2. 伊斯兰黄金时代
    1. 阿维森纳（伊本·西纳）、阿维罗伊（伊本·鲁世德）、阿尔-加扎利
    2. 保存并注释亚里士多德，发展出更精致的形而上学和医学。
3. 中国：儒学复兴
    1. 汉代经学 → 魏晋玄学 → 隋唐佛教 → 宋明理学（周敦颐、程颢、程颐、朱熹）、心学（陆九渊、王阳明）

# 近代哲学（17–18世纪）
1. 理性主义：笛卡尔（“我思故我在”、天赋观念）→ 斯宾诺莎 → 莱布尼茨  
2. 经验主义：培根、霍布斯、洛克（“白纸说”）→ 贝克莱 → 休谟（因果关系只是习惯）  
3. 启蒙运动与综合：康德（《纯粹理性批判》：先验综合判断，现象与物自体）
    1. 康德被称为“近代哲学的终结者和现代哲学的开启者”。

# 19世纪德国古典哲学
1. 费希特 → 谢林 → 黑格尔（绝对精神、辩证法、正-反-合）  
2. 叔本华（意志哲学，悲观主义，受印度哲学影响）  
3. 马克思（在黑格尔辩证法基础上建立历史唯物主义）  
4. 尼采（“上帝已死”、权力意志、超人、重估一切价值）

# 20世纪哲学：两大主流与多元化
1. 分析哲学（英美）
    1. 弗雷格、罗素、摩尔 → 早期维特根斯坦（《逻辑哲学论》）→ 逻辑实证主义（维也纳学圈）→ 后期维特根斯坦（语言游戏）→ 蒯因、克里普克、普特南
    2. 关注语言、逻辑、科学哲学，强调清晰性和可验证性。
2. 大陆哲学
    1. 现象学（胡塞尔 → 海德格尔《存在与时间》→ 萨特、梅洛-庞蒂）
    2. 存在主义（克尔凯郭尔、尼采 → 海德格尔、萨特、加缪）
    3. 批判理论（法兰克福学派：霍克海默、阿多诺、马尔库塞、哈贝马斯）
    4. 结构主义（索绪尔 → 列维-斯特劳斯）→ 后结构主义（福柯、德里达、德勒兹）
    5. 后现代主义（利奥塔、鲍德里亚）
